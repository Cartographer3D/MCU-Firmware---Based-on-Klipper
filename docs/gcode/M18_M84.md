# M18/M84 - Disable steppers

This command disables all steppers.


### Usage

``` M18 ```

| Argument           | Description |
| ------------------ | ----------- |
| ```[E<flag>]```    | E Disable |
| ```[S<seconds>]``` | Inactivity Timeout. If none specified, disable now. |
| ```[X<flag>]```    | X Disable |
| ```[Y<flag>]```    | Y Disable |
| ```[Z<flag>]```    | Z Disable |


### Example

Disable all steppers immediately

```M18```
