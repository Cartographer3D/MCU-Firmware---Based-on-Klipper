# M105 - Report Temperatures

Request a temperature report to be sent to the host at some point in the future.


### Usage

``` M105 [T<index>] ```

| Argument          | Description  |
| ----------------- | ------------ |
| ```[T<index>] ``` | Hotend index |


### Example

Get a temperature report.

```
M105
```
